A professional, responsive weather application that provides real-time meteorological data, interactive mapping, and data visualization. 


## ğŸš€ Key Features

* **Real-Time Data:** Fetches precise weather data using the **Open-Meteo API** (No API key required).
* **Interactive Map:** Integrated **Leaflet.js** to dynamically pinpoint locations and visualize the geography of the searched city.
* **Data Visualization:** Uses **Chart.js** to render a beautiful 24-hour temperature trend graph with gradient fills.
* **Smart Search:** Implemented **Debouncing** logic for the search bar to provide city auto-suggestions without spamming the API.
* **Detailed Metrics:** Displays UV Index, Wind Status, Humidity, Visibility, and Air Quality with dynamic status labels (e.g., "Poor", "Good").
* **Dynamic Theming:** Fully functional **Dark/Light Mode** toggle using CSS variables for smooth transitions.
* **Responsive Design:** Grid and Flexbox layout that adapts perfectly from Desktop dashboards to Mobile screens.
